Taco Stand

A simple uploading tool.


Example use, with curl

curl -F "file=@/my/file/location" {website}

Which will return a string with the new file location, ie:

https://website/spry-pumpkin.jpg

You can also upload a file that will only be downloadable once by doing the following:

curl -F "file=@/my/file/location" {website}/once
